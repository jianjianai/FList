# FList - 一个简洁的在线文件列表
将 GitHub Releases 以类似网盘的形式展示在网页上，方便用户下载开源软件。
支持视频、音频、图片、PDF 等文件的在线预览。

## 为什么选择 FList
- **轻量**：静态页面生成，无需数据库，无需服务器，Cloudflare Pages、Github Pages、Vercel、Netlify等都可以轻松部署，完全免费。
- **便捷**：支持视频、音频、图片、PDF 文件的在线预览。
- **开箱即用**：只需简单配置，即可开箱即用，无需繁琐操作。
